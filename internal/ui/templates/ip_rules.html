Adaptive enforcement

Scales the global rate limit and can force a bot challenge based on each client's current threat score (shown per row above): tighter for high-risk IPs, relaxed for low-risk ones. Reversible: every request re-checks the live score, so this never permanently blocks anyone the way Automatic banning does.
